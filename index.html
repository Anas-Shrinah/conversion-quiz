Number base conversion quiz

This quiz provides a range of base conversion exercises to help you understand different bases and how to translate between them. There are several difficulty levels, as well as an infinite mode, that runs forever, calculating a continuous percentage score.

If you get an answer wrong, the correct answer will be given.

Remember that sometimes, converting to an intermediate base may help make a conversion easier.
